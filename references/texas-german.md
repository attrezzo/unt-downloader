# Texas German Vocabulary and Orthography Reference

Reference for Texas German dialect features, English loanwords, and pre-1901
German orthography found in German-Texan newspapers (1880s-1900s).

---

## Decision Tree: Standard German vs. Texas German vs. OCR Error

When encountering an unusual word:

1. **Is it a known Texas German term?** → Keep as-is (see vocabulary below)
2. **Is it an English loanword used in Texas German?** → Keep as-is
3. **Is it valid pre-1901 German spelling?** → Keep as-is (see period spellings)
4. **Does applying Fraktur error correction produce a valid word?** → Correct it
5. **None of the above?** → Likely OCR error; correct to nearest valid word

## Pre-1901 German Orthography

The Second Orthographic Conference of 1901 standardized German spelling.
Newspapers from the 1890s use the older forms. **Do NOT modernize these.**

| Period form (keep) | Modern form (do NOT use) |
|-------------------|------------------------|
| thun | tun |
| Thür, Thüre | Tür |
| Theil | Teil |
| Thal | Tal |
| Thier | Tier |
| Thon | Ton |
| Thor | Tor |
| Noth | Not |
| nothwendig | notwendig |
| Rath | Rat |
| Rath geben | Rat geben |
| giebt | gibt |
| gieng | ging |
| seyn | sein (occasionally) |
| Sey | Sei |
| bey | bei (occasionally) |
| sey es | sei es |
| Litteratur | Literatur |
| Commission | Kommission |
| Correspondenz | Korrespondenz |
| Telegraphie | Telegrafie |
| Photographie | Fotografie |
| Bureau | Büro |
| Redacteur | Redakteur |
| Fabrick | Fabrik |
| Circulare | Zirkulare |
| Comptoir | Kontor |

### Spelling conventions to preserve

- Double consonants where modern German uses single: Litteratur, Committe
- th- where modern German uses t-: Theil, Thal, Thier
- -ey where modern German uses -ei: (occasional, depends on word)
- -c- where modern German uses -k-: Redacteur, Fabrick
- -ph- where modern German uses -f-: Telegraphie, Photographie
